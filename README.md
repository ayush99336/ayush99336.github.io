# Jekyll Site Starter

## Run locally

1. Install Ruby (3.0+ recommended) and Bundler.
2. Install gems:

   ```bash
   bundle install
   ```

3. Start the dev server:

   ```bash
   bundle exec jekyll serve
   ```

4. Open the local URL shown in the terminal (usually `http://127.0.0.1:4000`).

## Customize

- Site settings: `_config.yml`
- Home page: `index.md`
- About page: `about.md`
- Posts: `_posts/*.md`
- Styles: `assets/css/style.css`
